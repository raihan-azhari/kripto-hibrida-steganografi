# Panduan Upload Source Code ke GitHub

1. Buka GitHub dan buat repository baru, misalnya `kripto-hibrida-steganografi`.
2. Ekstrak ZIP project ini.
3. Buka terminal pada folder project.
4. Jalankan:

```bash
git init
git add .
git commit -m "Program kriptografi hibrida steganografi"
git branch -M main
git remote add origin https://github.com/USERNAME/kripto-hibrida-steganografi.git
git push -u origin main
```

5. Salin link repository GitHub dan masukkan link tersebut pada laporan bila diminta.
